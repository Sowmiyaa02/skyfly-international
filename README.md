# SkyFly International — Premium Visa Consultancy Website

A premium, futuristic, fully-responsive marketing website for **SkyFly International Pvt. Ltd.**, a professional overseas documentation and global visa guidance consultancy based in Trichy, Tamil Nadu, India.

## Technologies

- **TanStack Start** (React 19 + TanStack Router)
- **Tailwind CSS v4**
- **Framer Motion** — scroll-triggered animations, layout transitions, hover effects
- **TypeScript**
- **Vite** (build tool)
- **Netlify** (hosting + deployment)

## Key Features

- Cinematic hero section with typewriter animation, floating airplanes, radar pulse, globe, and departure board
- 43+ country cards with region filtering, lazy images, and Apply Now modal
- Animated counter stats, glassmorphism cards, trust bar marquee
- FAQ accordion, testimonial carousel, contact form with WhatsApp redirect
- Sticky navbar with active section tracking
- Floating WhatsApp button with pulse animation
- SEO-optimized head tags
- Fully responsive (mobile, tablet, desktop)

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
